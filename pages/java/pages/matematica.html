Operações Matemáticas em Java

Guia completo sobre operadores aritméticos, relacionais, lógicos e a classe Math

1. Introdução às Operações Matemáticas

As operações matemáticas são fundamentais em programação. Java oferece uma ampla gama de operadores e funções matemáticas que permitem realizar cálculos simples e complexos.

Neste guia, você aprenderá sobre os operadores aritméticos básicos, operadores de atribuição compostos, operadores relacionais e lógicos, além de explorar a poderosa classe Math do Java.

2. Operadores Aritméticos Básicos

Java possui cinco operadores aritméticos principais para realizar operações matemáticas:

+

Adição

Soma dois valores

-

Subtração

Subtrai um valor de outro

*

Multiplicação

Multiplica dois valores

/

Divisão

Divide um valor por outro

%

Módulo (Resto)

Retorna o resto da divisão

Exemplos de Operadores Aritméticos:
java 
public class OperadoresAritmeticos {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;
        
        // Operações básicas
        int soma = a + b;           // 13
        int subtracao = a - b;      // 7
        int multiplicacao = a * b;  // 30
        int divisao = a / b;        // 3 (divisão inteira)
        int resto = a % b;          // 1 (resto da divisão)
        
        System.out.println("Soma: " + soma);
        System.out.println("Subtração: " + subtracao);
        System.out.println("Multiplicação: " + multiplicacao);
        System.out.println("Divisão: " + divisao);
        System.out.println("Resto: " + resto);
        
        // Divisão com ponto flutuante
        double divisaoDecimal = (double) a / b;  // 3.333...
        System.out.println("Divisão decimal: " + divisaoDecimal);
    }
}

Atenção à Divisão Inteira!

Quando você divide dois números inteiros em Java, o resultado também é um inteiro (a parte decimal é descartada).

Exemplo: 10 / 3 = 3 (não 3.333...)

Para obter resultado decimal, converta pelo menos um dos valores para double:

(double) 10 / 3 = 3.333...

3. Operadores de Incremento e Decremento

Java oferece operadores especiais para aumentar ou diminuir o valor de uma variável em 1:

Operador Nome Descrição Exemplo
++ Incremento Aumenta o valor em 1 x++ ou ++x
-- Decremento Diminui o valor em 1 x-- ou --x
java 
public class IncrementoDecremento {
    public static void main(String[] args) {
        int x = 5;
        
        // Pós-incremento (usa primeiro, depois incrementa)
        System.out.println("x = " + x);      // 5
        System.out.println("x++ = " + x++);  // 5 (exibe 5, depois vira 6)
        System.out.println("x = " + x);      // 6
        
        // Pré-incremento (incrementa primeiro, depois usa)
        x = 5;
        System.out.println("++x = " + ++x);  // 6 (incrementa para 6, depois exibe)
        System.out.println("x = " + x);      // 6
        
        // Pós-decremento
        x = 5;
        System.out.println("x-- = " + x--);  // 5 (exibe 5, depois vira 4)
        System.out.println("x = " + x);      // 4
        
        // Pré-decremento
        x = 5;
        System.out.println("--x = " + --x);  // 4 (decrementa para 4, depois exibe)
        System.out.println("x = " + x);      // 4
    }
}

4. Operadores de Atribuição Compostos

Estes operadores combinam uma operação aritmética com atribuição, tornando o código mais conciso:

Operador Equivalente a Exemplo
+= x = x + valor x += 5
-= x = x - valor x -= 3
*= x = x * valor x *= 2
/= x = x / valor x /= 4
%= x = x % valor x %= 3
java 
public class OperadoresCompostos {
    public static void main(String[] args) {
        int saldo = 100;
        
        System.out.println("Saldo inicial: " + saldo);
        
        saldo += 50;  // Mesmo que: saldo = saldo + 50
        System.out.println("Após depósito de 50: " + saldo);  // 150
        
        saldo -= 30;  // Mesmo que: saldo = saldo - 30
        System.out.println("Após saque de 30: " + saldo);     // 120
        
        saldo *= 2;   // Mesmo que: saldo = saldo * 2
        System.out.println("Após dobrar: " + saldo);          // 240
        
        saldo /= 3;   // Mesmo que: saldo = saldo / 3
        System.out.println("Após dividir por 3: " + saldo);   // 80
        
        saldo %= 7;   // Mesmo que: saldo = saldo % 7
        System.out.println("Resto da divisão por 7: " + saldo); // 3
    }
}

5. Operadores Relacionais

Operadores relacionais comparam dois valores e retornam um resultado booleano (true ou false):

Operador Significado Exemplo Resultado
== Igual a 5 == 5 true
!= Diferente de 5 != 3 true
> Maior que 7 > 5 true
< Menor que 3 < 8 true
>= Maior ou igual a 5 >= 5 true
<= Menor ou igual a 4 <= 6 true
java 
public class OperadoresRelacionais {
    public static void main(String[] args) {
        int idade = 18;
        int limiteIdade = 18;
        
        System.out.println("idade == limiteIdade: " + (idade == limiteIdade));  // true
        System.out.println("idade != limiteIdade: " + (idade != limiteIdade));  // false
        System.out.println("idade > limiteIdade: " + (idade > limiteIdade));    // false
        System.out.println("idade < limiteIdade: " + (idade < limiteIdade));    // false
        System.out.println("idade >= limiteIdade: " + (idade >= limiteIdade));  // true
        System.out.println("idade <= limiteIdade: " + (idade <= limiteIdade));  // true
        
        // Uso prático em condicionais
        if (idade >= limiteIdade) {
            System.out.println("Pode votar!");
        }
    }
}

6. Operadores Lógicos

Operadores lógicos são usados para combinar múltiplas condições:

Operador Nome Descrição Exemplo
&& AND (E) Retorna true se ambas condições forem verdadeiras (x > 5) && (x < 10)
|| OR (OU) Retorna true se pelo menos uma condição for verdadeira (x < 5) || (x > 10)
! NOT (NÃO) Inverte o valor booleano !(x == 5)
java 
public class OperadoresLogicos {
    public static void main(String[] args) {
        int idade = 25;
        boolean temCarteira = true;
        
        // AND - ambas condições devem ser verdadeiras
        if (idade >= 18 && temCarteira) {
            System.out.println("Pode dirigir!");
        }
        
        // OR - pelo menos uma condição deve ser verdadeira
        boolean fimDeSemana = true;
        boolean feriado = false;
        
        if (fimDeSemana || feriado) {
            System.out.println("Não precisa trabalhar!");
        }
        
        // NOT - inverte o valor
        boolean chovendo = false;
        
        if (!chovendo) {
            System.out.println("Pode sair sem guarda-chuva!");
        }
        
        // Combinando operadores
        int nota = 85;
        int frequencia = 80;
        
        if (nota >= 70 && frequencia >= 75) {
            System.out.println("Aprovado!");
        }
    }
}

Tabela Verdade:

AND (&&):

  • true && true = true
  • true && false = false
  • false && true = false
  • false && false = false

OR (||):

  • true || true = true
  • true || false = true
  • false || true = true
  • false || false = false

7. A Classe Math

A classe Math do Java fornece métodos para operações matemáticas avançadas:

java 
public class ClasseMath {
    public static void main(String[] args) {
        // Constantes matemáticas
        System.out.println("PI: " + Math.PI);      // 3.141592653589793
        System.out.println("E: " + Math.E);        // 2.718281828459045
        
        // Potência e raiz
        double potencia = Math.pow(2, 3);          // 2^3 = 8.0
        double raiz = Math.sqrt(16);               // √16 = 4.0
        double raizCubica = Math.cbrt(27);         // ∛27 = 3.0
        
        System.out.println("2^3 = " + potencia);
        System.out.println("√16 = " + raiz);
        System.out.println("∛27 = " + raizCubica);
        
        // Valor absoluto
        int negativo = -15;
        int absoluto = Math.abs(negativo);
        System.out.println("Valor absoluto de -15: " + absoluto);  // 15
        
        // Arredondamento
        double numero = 3.7;
        long arredondado = Math.round(numero);      // 4
        double arredondadoCima = Math.ceil(3.2);    // 4.0 (arredonda para cima)
        double arredondadoBaixo = Math.floor(3.8);  // 3.0 (arredonda para baixo)
        
        System.out.println("Round de 3.7: " + arredondado);
        System.out.println("Ceil de 3.2: " + arredondadoCima);
        System.out.println("Floor de 3.8: " + arredondadoBaixo);
        
        // Máximo e Mínimo
        int max = Math.max(10, 20);    // 20
        int min = Math.min(10, 20);    // 10
        
        System.out.println("Máximo entre 10 e 20: " + max);
        System.out.println("Mínimo entre 10 e 20: " + min);
        
        // Número aleatório (0.0 a 1.0)
        double aleatorio = Math.random();
        System.out.println("Número aleatório: " + aleatorio);
        
        // Número aleatório entre 1 e 100
        int aleatorioInteiro = (int) (Math.random() * 100) + 1;
        System.out.println("Número aleatório entre 1 e 100: " + aleatorioInteiro);
    }
}

Principais Métodos da Classe Math:

  • Math.pow(base, expoente) - Potenciação
  • Math.sqrt(numero) - Raiz quadrada
  • Math.cbrt(numero) - Raiz cúbica
  • Math.abs(numero) - Valor absoluto
  • Math.round(numero) - Arredondamento
  • Math.ceil(numero) - Arredonda para cima
  • Math.floor(numero) - Arredonda para baixo
  • Math.max(a, b) - Retorna o maior valor
  • Math.min(a, b) - Retorna o menor valor
  • Math.random() - Número aleatório entre 0.0 e 1.0

8. Ordem de Precedência dos Operadores

Quando múltiplos operadores aparecem em uma expressão, Java segue uma ordem específica de avaliação:

Prioridade Operadores Descrição
1 (Mais alta) ( ) Parênteses
2 ++, -- Incremento e Decremento
3 *, /, % Multiplicação, Divisão, Módulo
4 +, - Adição, Subtração
5 <, <=, >, >= Operadores Relacionais
6 ==, != Igualdade e Desigualdade
7 && AND Lógico
8 || OR Lógico
9 (Mais baixa) =, +=, -=, *=, /=, %= Atribuição
java 
public class Precedencia {
    public static void main(String[] args) {
        // Sem parênteses - segue ordem de precedência
        int resultado1 = 10 + 5 * 2;
        System.out.println("10 + 5 * 2 = " + resultado1);  // 20 (multiplicação primeiro)
        
        // Com parênteses - altera a precedência
        int resultado2 = (10 + 5) * 2;
        System.out.println("(10 + 5) * 2 = " + resultado2);  // 30 (soma primeiro)
        
        // Exemplo complexo
        int resultado3 = 5 + 3 * 2 - 8 / 4;
        // Passo 1: 3 * 2 = 6
        // Passo 2: 8 / 4 = 2
        // Passo 3: 5 + 6 = 11
        // Passo 4: 11 - 2 = 9
        System.out.println("5 + 3 * 2 - 8 / 4 = " + resultado3);  // 9
        
        // Use parênteses para clareza
        int resultado4 = (5 + 3) * (2 - 8) / 4;
        System.out.println("(5 + 3) * (2 - 8) / 4 = " + resultado4);  // -12
    }
}

Dica Importante:

Use parênteses sempre que houver dúvida sobre a ordem de avaliação. Isso torna o código mais legível e evita erros.

9. Exemplos Práticos

Exemplo 1: Calculadora de IMC

java 
import java.util.Scanner;

public class CalculadoraIMC {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Digite seu peso (kg): ");
        double peso = sc.nextDouble();
        
        System.out.print("Digite sua altura (m): ");
        double altura = sc.nextDouble();
        
        // Cálculo do IMC: peso / (altura²)
        double imc = peso / Math.pow(altura, 2);
        
        // Arredonda para 2 casas decimais
        imc = Math.round(imc * 100.0) / 100.0;
        
        System.out.println("Seu IMC é: " + imc);
        
        // Classificação
        if (imc < 18.5) {
            System.out.println("Classificação: Abaixo do peso");
        } else if (imc < 25) {
            System.out.println("Classificação: Peso normal");
        } else if (imc < 30) {
            System.out.println("Classificação: Sobrepeso");
        } else {
            System.out.println("Classificação: Obesidade");
        }
        
        sc.close();
    }
}

Exemplo 2: Calculadora de Desconto

java 
import java.util.Scanner;

public class CalculadoraDesconto {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Digite o valor do produto: R$ ");
        double valorOriginal = sc.nextDouble();
        
        System.out.print("Digite a porcentagem de desconto: ");
        double percentualDesconto = sc.nextDouble();
        
        // Calcula o valor do desconto
        double valorDesconto = valorOriginal * (percentualDesconto / 100);
        
        // Calcula o valor final
        double valorFinal = valorOriginal - valorDesconto;
        
        System.out.println("\n=== RESUMO ===");
        System.out.println("Valor original: R$ " + valorOriginal);
        System.out.println("Desconto (" + percentualDesconto + "%): R$ " + valorDesconto);
        System.out.println("Valor final: R$ " + valorFinal);
        System.out.println("Você economizou: R$ " + valorDesconto);
        
        sc.close();
    }
}

Exemplo 3: Conversor de Temperatura

java 
import java.util.Scanner;

public class ConversorTemperatura {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== CONVERSOR DE TEMPERATURA ===");
        System.out.println("1 - Celsius para Fahrenheit");
        System.out.println("2 - Fahrenheit para Celsius");
        System.out.print("Escolha uma opção: ");
        
        int opcao = sc.nextInt();
        
        if (opcao == 1) {
            System.out.print("Digite a temperatura em Celsius: ");
            double celsius = sc.nextDouble();
            
            // Fórmula: F = (C × 9/5) + 32
            double fahrenheit = (celsius * 9.0 / 5.0) + 32;
            
            System.out.println(celsius + "°C = " + fahrenheit + "°F");
            
        } else if (opcao == 2) {
            System.out.print("Digite a temperatura em Fahrenheit: ");
            double fahrenheit = sc.nextDouble();
            
            // Fórmula: C = (F - 32) × 5/9
            double celsius = (fahrenheit - 32) * 5.0 / 9.0;
            
            System.out.println(fahrenheit + "°F = " + celsius + "°C");
            
        } else {
            System.out.println("Opção inválida!");
        }
        
        sc.close();
    }
}

Exemplo 4: Jogo de Dados

java 
import java.util.Scanner;

public class JogoDados {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== JOGO DE DADOS ===");
        System.out.print("Pressione ENTER para jogar os dados...");
        sc.nextLine();

        int dado1 = (int) (Math.random() * 6) + 1;
        int dado2 = (int) (Math.random() * 6) + 1;
        
        System.out.println("Dado 1: " + dado1);
        System.out.println("Dado 2: " + dado2);
        
        int soma = dado1 + dado2;
        System.out.println("Soma: " + soma);
        
        // Verifica resultados especiais
        if (dado1 == dado2) {
            System.out.println("DUPLA! Você tirou dois " + dado1 + "s!");
        }
        
        if (soma == 7) {
            System.out.println("SORTE! A soma deu 7!");
        } else if (soma == 12) {
            System.out.println("JACKPOT! Soma máxima!");
        } else if (soma == 2) {
            System.out.println("AZAR! Soma mínima!");
        }
        
        sc.close();
    }
}
