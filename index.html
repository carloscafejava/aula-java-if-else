IF, ELSE e ELSE IF em Java

Na vida real voce faz decisões em sequência. Em código é a mesma coisa: testar uma condição, agir se for verdadeira e pular as demais quando uma for satisfeita.

Sintaxe basica : 
if (condição) {
    // executa se condição for verdadeira
} else if (outraCondicao) {
    // executa se a primeira for falsa e esta for verdadeira
} else {
    // executa se todas as anteriores forem falsas
}

Conceitos essenciais

IF Primeiro teste. Se verdadeiro, o restante da cadeia é ignorado.
ELSE IF Testes alternativos que só são avaliados se os anteriores falharem.
ELSE Plano B obrigatório — pega tudo que sobrou.

Operadores de comparação e lógicos

// Comparações
==    !=    >   <   >=   <=
// Lógicos
&& (AND), || (OR), ! (NOT)

== -> compara referências na memória. Ou seja, ele quer saber se as duas variáveis apontam pro mesmo objeto.

.equals() -> compara o conteúdo. No caso de String, compara o texto que está dentro dela.

Exemplo:


String a = new String("Carlos");
String b = new String("Carlos");

System.out.println(a == b);       // false -> não são o mesmo objeto na memória
System.out.println(a.equals(b));  // true  -> o conteúdo é igual

Outro caso:


String x = "Java";
String y = "Java";

System.out.println(x == y);       // true -> nesse caso sim, porque o compilador guarda no pool de Strings
System.out.println(x.equals(y));  // true  -> conteúdo igual também

Ou seja:
Sempre use .equals() pra comparar Strings, porque você está interessado no conteúdo e não em saber se as duas variáveis apontam pro mesmo espaço de memória.

O == até pode dar “true” às vezes (por causa do string pool), mas é sorte de principiante. Mais cedo ou mais tarde, vai dar ruim. 

if (obj != null && obj.isAtivo()) {
    // seguro: obj.isAtivo() só é chamado se obj != null
}

Boas práticas (leia se valoriza sua sanidade)

  • Sempre use chaves { } mesmo para uma única linha dentro do if — evita bugs quando alguém mexer depois.
  • Evite cadeias enormes de else if — considere switch, Map de funções ou polimorfismo.
  • Menos aninhamento: prefira guard clauses (retornos antecipados).
  • Extraia condições complexas para métodos com nome significativo (isUsuarioAtivo()).

Erros comuns que dão dor de cabeça

  • Usar == com Strings em vez de .equals().
  • Comparar doubles com == (use epsilon).
  • Esquecer que if pára na primeira condição verdadeira.
  • Não fechar Scanner: causa warnings e, às vezes, travamento no fluxo de entrada.

Exemplos completos (práticos)

Exemplo: Calculadora com menu (corrigida e completa)

import java.util.Scanner;

public class CalculadoraMenu {
    public static void main(String[] args) {
        Scanner teclado = new Scanner(System.in);

        System.out.print("Digite o primeiro número: ");
        double num1 = teclado.nextDouble();

        System.out.print("Digite o segundo número: ");
        double num2 = teclado.nextDouble();

        System.out.println("Escolha a operação:");
        System.out.println("1 - Somar");
        System.out.println("2 - Subtrair");
        System.out.println("3 - Multiplicar");
        System.out.println("4 - Dividir");
        int opcao = teclado.nextInt();

        if (opcao == 1) {
            System.out.println("Resultado: " + (num1 + num2));
        } else if (opcao == 2) {
            System.out.println("Resultado: " + (num1 - num2));
        } else if (opcao == 3) {
            System.out.println("Resultado: " + (num1 * num2));
        } else if (opcao == 4) {
            if (num2 != 0) {
                System.out.println("Resultado: " + (num1 / num2));
            } else {
                System.out.println("Erro: divisão por zero!");
            }
        } else {
            System.out.println("Opção inválida.");
        }

        teclado.close();
    }
}

Exemplo: Comparação de Strings corretamente

import java.util.Scanner;

public class CompararStrings {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Digite a senha: ");
        String senha = sc.nextLine();

        if ("admin".equals(senha)) {
            System.out.println("Acesso permitido");
        } else {
            System.out.println("Acesso negado");
        }

        sc.close();
    }
}

Exemplo: Switch vs If (quando usar switch)

int dia = 3;
switch (dia) {
    case 1: System.out.println("Segunda"); break;

    case 3: System.out.println("Quarta"); break;

}

Refatoração de menu com Map (padrão mais moderno)

import java.util.HashMap;
import java.util.Map;
import java.util.Scanner;

public class MenuMap {
    public static void main(String[] args) {
        Map<Integer,Runnable> acoes = new HashMap<>();
        acoes.put(1, () -> System.out.println("Somar..."));
        acoes.put(2, () -> System.out.println("Subtrair..."));
        acoes.put(3, () -> System.out.println("Multiplicar..."));
        acoes.put(4, () -> System.out.println("Dividir..."));

        Scanner sc = new Scanner(System.in);
        System.out.print("Opção: ");
        int opc = sc.nextInt();
        Runnable acao = acoes.get(opc);
        if (acao != null) {
            acao.run();
        } else {
            System.out.println("Opção inválida.");
        }
        sc.close();
    }
}

Atividades por tópico (com soluções escondidas)

Sintaxe básica

1) Dado um inteiro nota, imprima "Aprovado" se >= 7, caso contrário imprima "Reprovado".

Mostrar solução 
int nota = 6;
if (nota >= 7) {
    System.out.println("Aprovado");
} else {
    System.out.println("Reprovado");
}
Operadores lógicos

2) Verifique se uma pessoa pode votar: condição = (idade >= 18) e (não está suspensa). Use booleanos fictícios.

Mostrar solução 
int idade = 20;
boolean suspensa = false;
if (idade >= 18 && !suspensa) {
    System.out.println("Pode votar");
} else {
    System.out.println("Não pode votar");
}
Curto-circuito e null

3) Dada uma String usuario, imprima "Bem-vindo" somente se usuario não for nulo e tiver mais de 3 caracteres.

Mostrar solução 
String usuario = "carlos";
if (usuario != null && usuario.length() > 3) {
    System.out.println("Bem-vindo, " + usuario);
} else {
    System.out.println("Usuário inválido");
}
Comparação de Strings

4) Leia uma entrada e verifique se é igual a "exit" (ignorar maiúsculas).

Mostrar solução 
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
System.out.print("Comando: ");
String cmd = sc.nextLine();
if ("exit".equalsIgnoreCase(cmd)) {
    System.out.println("Encerrando...");
} else {
    System.out.println("Comando diferente");
}
sc.close();
Ternário

5) Transforme o código if/else abaixo em um operador ternário:
if (x % 2 == 0) tipo = "par"; else tipo = "ímpar";

Mostrar solução 
int x = 7;
String tipo = (x % 2 == 0) ? "par" : "ímpar";
System.out.println(tipo);
Switch vs If

6) Dado um número 1-7 imprima o dia da semana usando switch. Depois reescreva com if/else (atividade opcional).

Mostrar solução (switch) 
int d = 5;
switch (d) {
    case 1: System.out.println("Segunda"); break;
    case 2: System.out.println("Terça"); break;
    case 3: System.out.println("Quarta"); break;
    case 4: System.out.println("Quinta"); break;
    case 5: System.out.println("Sexta"); break;
    case 6: System.out.println("Sábado"); break;
    case 7: System.out.println("Domingo"); break;
    default: System.out.println("Número inválido"); break;
}
Guardar cláusulas (guard clauses)

7) Escreva um método que recebe idade e retorna true se é maior de idade; use retorno antecipado (guard clause) para idade negativa.

Mostrar solução 
public static boolean isAdult(int idade) {
    if (idade < 0) return false; // entrada inválida
    return idade >= 18;
}
Comparação de ponto flutuante

8) Compare dois doubles a e b com precisão (epsilon).

Mostrar solução 
double a = 0.1 + 0.2;
double b = 0.3;
double eps = 1e-9;
if (Math.abs(a - b) < eps) {
    System.out.println("Iguais (próximo o suficiente)");
} else {
    System.out.println("Diferentes");
}
Refatoração: menu com Map

9) Substitua um grande if/else que chama funções diferentes por um Map<Integer,Runnable> (exercício de refatoração).

Mostrar solução 
Map<Integer,Runnable> acoes = new java.util.HashMap<>();
acoes.put(1, () -> System.out.println("Ação 1"));
acoes.put(2, () -> System.out.println("Ação 2"));
// ...
int opc = 2;
Runnable r = acoes.get(opc);
if (r != null) r.run(); else System.out.println("Inválida");
